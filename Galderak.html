Galdera:

Zeintzuk dira werbitzari web bat etxean prestatzeko beharrezko osagaiak?

Erantzuna:

   Hardware osagaiak: Gutxienez behar izango ditugu, Interneteko konexioarekiko bideratzailea (router), zerbitzari weba eta RJ45 sare kableatzea bideratzailea gure ordenagailuarekin konektatzeko ordezkatzen dituen ordenagailua.
   Osagai softwareak: Windows Sistema Eragileak edo Linux-ek eta zerbitzaria programatzen duen Apache Server. Web-orri-edizioa egin ahal izateko: Dreamweaver (ordainketako) edo Joomla, WordPress (doakoa), VS Code...
