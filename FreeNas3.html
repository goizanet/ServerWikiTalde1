HARDWARE BALDINTZAK INSTALAZIOA INST. PROXMOX SEGURTASUN KOPIAK

FREENAS

LEHEN PAUSOAK

Hasierako baldintzak instalazioa hasteko

FreeNAS-eko webgunetik azken “imagen” jaitsiko dugu:

https://www.freenas.org/download-freenas-release/

Hasierako instalazioa egin ahal izateko monitore bat beharko dugu (instalazioaren ondoren hau deskonektatu ahalko da).

Instalazioa egin eta gero, beste ordenagailu bat beharko dugu internetarekin GUI-ra sartu ahal izateko..

Behin “imagen”-a deskargatu dugula hutsik dagoen USB edo diska batean sartuko dugu.

Instalazioa - Zerbitzari Fisikoan

  1.   Aukeratutako makinan USB/Diska sartuko dugu gure “device” eta FreeNAS-eko “imagen”-arekin.
  2.   BIOS-era sartu, eta sistema boot-device-a FreeNAS-ekoa aukeratu egingo dugu.
  3.   Sistema berrabiarazi.
  4.   FreeNAS-eko instalazio menua pantailan agerraraztean lehenengo aukera (1) aukeratuko dugu instalazioa hasteko.
  5.   Kontsolako ‘Setup menu’-a kargatzen zaigunean, Install/Upgrade aukeratu.
  6.   Hurrengo menuan FreeNAS erabiltzeko ze drive erabiliko den aukeratzeko eskatuko digute. Hemen boot device-a aukeratuko dugu, kontuz, ez “storage disk”-a. Orokorrean boot device-a storagekoa baino tamaina txikiagoa izango du. Nabigatzeko Aukeratzeko espazio barra sakatu. Ondoren OK sakatu.
  7.   Ondorengo pantailan, “Yes” sakatu instalazioarekin jarraitu ahal izateko.
  8.   Jarraian FreeNAS-era sartzeko erabiliko dugun pasahitza sartzeko eskatuko digute.
  9.   FreeNAS Boot Mode menu pantailarekin topatzen garenean Boot via BIOS aukeratuko dugu (orokorrean ‘motherboard’ guztiekin funtzionatzen baitu eta Hardware zaharragoarekin be).
  10.   Gure instalazioa hasiko da, eta minutu gutxi batzuk beharko ditu hau bukatzeko.
  11.   OK sakatu sistema berrabiarazteko eta gero instalazioa egiteko erabili dugun diska edo USB-a kenduko dugu. Adi, sistemak bi aldiz galdetu egingo dizu diska edo USB-a kendu izan duzun edo ez.
  12.   Sistema berrabiaraziko da Boot device-tik.
  13.   FreeNAS kargatzen den lehen aldia denez, mezuak agertuko dira pantaila osoan zehar kargatzen den bitartean.
  14.   Amaitzen denean Console Setup-arekin topatuko gara IP helbide batekin batera (azken hau 0.0.0.0. baldin bada, interneteko kablea konektatuta dagoela eta DHPC zerbitzari bat duela ziurtatu).
  15.   Orain beste ordenagailu batean web-arakatzailearen bilatzailean IP helbide hori bilatuko dugu, eta jarraian FreeNAS-eko login menuarekin topatu beharko ginen. “root” erabiltzeailearekin logueatu goiko pausu batean sartu duzun pasahitzarekin.
  16.   LISTO! FreesNAS instalatu berri duzu.
