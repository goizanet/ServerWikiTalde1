APACHE ETA MYSQL

DESKRIBAPENA

  Apache HTTP Server doako web-zerbitzariaren software bat da, Unix plataformetarako kode irekikoa, eta harekin mundu osoko webguneen %46 exekutatzen da. Webguneen jabeei web-eko edukia zerbitzatzeko aukera ematen die.


NOLA EGINGO DUGU

  1. Apache: Web serbitzariak sortzeko, horretarako terminalean hurrengo komandoak idatziko ditugu, sistemaren eguneraketarekin batera: 
    2.                                     $ sudo apt update
                                    $ sudo apt install apache2
  2. MySQL: Datu basetara konektatu ahal izateko, aurretik egin dugun bezala, terminalean hurrengo koamndoak idatzi behar ditugu: 
    3.                                     $ sudo apt install mysql-server
