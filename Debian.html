DEBIAN

ZER DA?

  GNU / Linuxen distribuzioetako sistema eragile bat da, ezagunenetarikoa, egonkorrenetakoa eta zaharrenetarikoa, gainera, software libre batean dago oinarrituta. Ubuntu bezalako distribuzio ezagunak sistema eragile honetan oinarritu zen etaedozein lekutan erabili daiteke, bai ordenagailuan, etxean zerbitzari batean, bulegoan, eta fabrikatzailean baita ere.
